Building Insights: A Data Journey Through LEGO Sets

I’m currently working on a project where I analyze and visualize a dataset from Kaggle.com using Microsoft Power BI. The data is about LEGO sets. These are the techniques I’m using to tell a compelling data story:

  • Exploratory Data Analysis with Power BI
  • Data Visualization with Power BI
  • Regression Analysis with Power BI
  • Time Series Exploration with Power BI
  • Clustering Techniques using Python

This project is about using the data to tell a story about the LEGO data universe. It not only demonstrates my technical skills, but also showcases my ability to derive meaningful insights from complex datasets.
